Validation

I have used websites in HTML5 and CSS before, but I was not never creating the website that has interactive forms, as well as navigation bar. I was always passionate on how one website can change different peoples lives by reading its content. I like to play around with the code and watch my site evolve from a basic structure to a website that is fully interactive with an eye catchy interface.

Creating this website was quite challenging as I never used any website validation to check if my website is free from any errors. I was surprised on how many mistakes I have made in the past in HTML and CSS code.

I feel that I have learned a lot about the website structure as well as how to close every class correctly. I enjoy making websites even more now as I know that there is way more things I can be challenged with. My aim as a web designer I am becoming is to be able to write excellent codes with no need to use any validation tools, but I know that this process will take some time to achieve.

My website is fully interactive that work on all screen sizes devices without losing any of its functionalities.

I have explored many websites before I have created this one to have an idea of how I want to create this one that will look great and stand out more from the other websites that I have seen.
